🎤 음성 기반 키오스크 AI 주문 시스템

식당에서의 자연어 음성 주문 처리 시스템

🎯 프로젝트 개요

기존 터치형 키오스크의 한계를 극복하고, 완전히 음성으로만 구동되는 키오스크 서비스를 구현했습니다. 사용자의 음성 입력을 분석하여 주문, 변경, 취소, 결제 등의 의도를 파악하고 음성으로 응답을 제공합니다.

🎤 음성 전처리

다중화자인식/구분을 통한 노이즈 개선

🗣️ 음성 인식

Whisper 기반 모델 파인튜닝

🧠 대화 처리

GPT-4o 활용 멀티턴 대화

📋 주문 관리

주문 추가, 변경, 취소, 결제

🏗️ 시스템 아키텍처

👤 사용자 음성 입력
🎤 마이크 캡처 & VAD 처리
🔊 음성 전처리 (노이즈 제거, 화자 분리)
🗣️ Whisper 기반 음성-텍스트 변환
🧠 GPT-4o 의도 파악 및 분석
📋 비즈니스 로직 처리 (주문 관리)
📝 자연어 응답 생성
🔊 TTS 음성 응답 출력

⏱️ 전체 처리 시간: 약 20초 (실측, 네트워크 포함)
